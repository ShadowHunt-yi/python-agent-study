# 07 — 生成器与迭代器

## 一、迭代器协议

```python
# 可迭代对象 —— 有 __iter__ 方法
# 迭代器 —— 有 __iter__ 和 __next__ 方法

# 手动迭代
nums = [1, 2, 3]
it = iter(nums)       # 获取迭代器
next(it)              # 1
next(it)              # 2
next(it)              # 3
next(it)              # StopIteration

# for 循环内部就是调用 iter() 和 next()
for x in nums:
    print(x)
```

## 二、生成器函数

用 `yield` 代替 `return`，每次调用 `next()` 执行到下一个 `yield`。

```python
def countdown(n):
    """从 n 倒数到 1"""
    while n > 0:
        yield n
        n -= 1

for num in countdown(5):
    print(num)    # 5, 4, 3, 2, 1

# 手动迭代
gen = countdown(3)
next(gen)   # 3
next(gen)   # 2
next(gen)   # 1
next(gen)   # StopIteration
```

```javascript
// JS 对比 —— function* + yield
function* countdown(n) {
    while (n > 0) {
        yield n--;
    }
}
```

## 四、yield from —— 委托生成器

```python
def flatten(nested):
    """展平嵌套列表"""
    for item in nested:
        if isinstance(item, list):
            yield from flatten(item)   # 委托给子生成器
        else:
            yield item

list(flatten([1, [2, [3, 4], 5], 6]))
# [1, 2, 3, 4, 5, 6]
```

```python
# 链接多个可迭代对象
def chain(*iterables):
    for it in iterables:
        yield from it

list(chain([1, 2], [3, 4], [5, 6]))
# [1, 2, 3, 4, 5, 6]
```

## 六、管道模式

```python
# 生成器可以像 Unix 管道一样组合
def read_lines(path):
    """读取文件行"""
    with open(path, "r", encoding="utf-8") as f:
        for line in f:
            yield line.strip()

def filter_comments(lines):
    """过滤注释行"""
    for line in lines:
        if not line.startswith("#"):
            yield line

def filter_empty(lines):
    """过滤空行"""
    for line in lines:
        if line:
            yield line

def parse_csv(lines):
    """解析 CSV"""
    for line in lines:
        yield line.split(",")

# 组合成管道 —— 全程惰性，内存高效
lines = read_lines("data.csv")
non_comments = filter_comments(lines)
non_empty = filter_empty(non_empty)
records = parse_csv(non_empty)

for record in records:
    process(record)
```

## 七、itertools —— 标准库工具

```python
import itertools

# count —— 无限计数
for i in itertools.count(10):    # 10, 11, 12, ...
    if i > 15: break

# cycle —— 无限循环
for item in itertools.cycle(["a", "b", "c"]):
    print(item)    # a, b, c, a, b, c, ...

# chain —— 链接多个迭代器
list(itertools.chain([1, 2], [3, 4]))    # [1, 2, 3, 4]

# islice —— 切片生成器
list(itertools.islice(itertools.count(), 5))    # [0, 1, 2, 3, 4]

# groupby —— 分组
data = [("a", 1), ("a", 2), ("b", 3), ("b", 4)]
for key, group in itertools.groupby(data, key=lambda x: x[0]):
    print(key, list(group))

# product —— 笛卡尔积
list(itertools.product([1, 2], ["a", "b"]))
# [(1, 'a'), (1, 'b'), (2, 'a'), (2, 'b')]

# permutations —— 排列
list(itertools.permutations([1, 2, 3], 2))
# [(1, 2), (1, 3), (2, 1), (2, 3), (3, 1), (3, 2)]

# combinations —— 组合
list(itertools.combinations([1, 2, 3], 2))
# [(1, 2), (1, 3), (2, 3)]
```

## 九、生成器的双向通信：send / throw / close

```python
# send() —— 向生成器发送值，成为 yield 表达式的返回值
def running_average():
    total = 0
    count = 0
    average = None
    while True:
        value = yield average      # yield 返回 average，send 进来的值赋给 value
        if value is None:
            break
        total += value
        count += 1
        average = total / count

gen = running_average()
next(gen)           # 启动生成器，执行到第一个 yield（average=None）
gen.send(10)        # value=10, 返回 10.0
gen.send(20)        # value=20, 返回 15.0
gen.send(30)        # value=30, 返回 20.0
```

```python
# throw() —— 在 yield 处抛入异常
def my_gen():
    try:
        yield 1
        yield 2
    except ValueError:
        yield "caught error"
    yield 3

gen = my_gen()
next(gen)              # 1
gen.throw(ValueError)  # "caught error"（异常被生成器内部捕获）
next(gen)              # 3
```

```python
# close() —— 在 yield 处抛入 GeneratorExit，终止生成器
def my_gen():
    try:
        yield 1
        yield 2
    except GeneratorExit:
        print("生成器被关闭了")
        # 注意：GeneratorExit 中不能 yield，只能清理后退出

gen = my_gen()
next(gen)    # 1
gen.close()  # 打印 "生成器被关闭了"
```

### yield 的三种角色

```python
def example():
    # yield 有三种角色，取决于上下文：
    output = yield input_value
    #        ↑         ↑
    #        |         └─ 1. 向外返回值（next/send 调用方收到）
    #        └─────────── 2. 从外部接收值（send 传入的值）
    #                    3. 接收 throw 抛入的异常

    # 简单记忆：
    # - next(gen)  → 执行到 yield，返回 yield 后面的值
    # - gen.send(x) → 执行到 yield，x 成为 yield 表达式的结果，返回下一个 yield 的值
    # - gen.throw(E) → 在 yield 处抛入异常 E
```

## 十、生成器 vs async 迭代器

```python
# 同步生成器
def sync_gen():
    yield 1
    yield 2

# 异步生成器（Agent 流式输出用这个）
async def async_gen():
    yield 1
    await asyncio.sleep(0.1)
    yield 2

# 消费方式不同
for x in sync_gen():           # 同步 for
    print(x)

async for x in async_gen():   # 异步 for（Python 3.6+）
    print(x)

# 异步生成器没有 send()，只有 __anext__()
```

### Agent 流式输出的实际模式

# OpenAI SDK 流式 + FastAPI SSE
async def stream_agent_response(query: str):
    """Agent 流式输出（真实模式）"""
    messages = [{"role": "user", "content": query}]

    # 真正的流式调用 OpenAI
    stream = await client.chat.completions.create(

## 十一、面试高频问题

**Q: 生成器和列表推导式的区别？**
A: 生成器惰性求值、省内存、只能遍历一次。列表推导式立即求值、可重复遍历。

**Q: `yield` 和 `return` 的区别？**
A: `return` 终止函数并返回值。`yield` 暂停函数并返回值，下次调用 `next()` 从暂停处继续。

**Q: `send()` 和 `next()` 的区别？**
A: `next(gen)` 只恢复执行，`gen.send(value)` 恢复执行并把 `value` 作为 `yield` 表达式的结果。第一次必须用 `next(gen)` 或 `gen.send(None)` 启动。

**Q: 异步生成器和同步生成器的区别？**
A: 异步生成器用 `async def` + `yield`，用 `async for` 消费，每次 yield 之间可以 `await`。Agent 流式输出必须用异步生成器。

**Q: `yield from` 的作用？**
A: 将迭代委托给子生成器。简化嵌套 yield，如 `yield from flatten(item)` 等价于 `for x in flatten(item): yield x`。

**Q: 如何在 Agent 开发中使用生成器？**
A: 流式 LLM 输出（`async for chunk in stream`）、大数据集处理、批量 API 调用、管道式数据处理。
