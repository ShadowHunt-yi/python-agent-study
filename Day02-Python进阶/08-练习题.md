```python
# 4.2 实现一个分页数据获取器
def fetch_all_pages(base_url: str, page_size: int = 10):
    # 模拟分页 API，每次返回一页数据
    # 当返回空列表时停止
    import random
    page = 1
    while True:
        # 模拟 API 调用
        data = [f"item_{(page-1)*page_size + i}" for i in range(page_size)]
        if page > 3:    # 模拟只有 3 页
            data = []
        if not data:
            break
        yield from data
        page += 1

items = list(fetch_all_pages("http://api.example.com"))
assert len(items) == 30
```

## 答案提示

1. Stack：用 list 实现，`append`/`pop` 就是栈操作
2. ShoppingCart：用 `next()` 找已存在的商品
3. memoize：用 dict 存储 `{args: result}`
4. sliding_window：用 `deque(maxlen=size)` 或切片
