# 01 — Mini Agent 项目结构

## 目录结构

```
mini-agent/
├── pyproject.toml            # 类比 package.json（依赖 + 构建配置）
├── README.md
├── .env.example
├── src/                      # src 布局：避免 import 歧义（import 的是安装后的包，不是本地目录）
│   └── mini_agent/
│       ├── __init__.py       # 标记为 Python 包（类似 Node.js 的 package.json，但可以为空）
│       ├── main.py           # FastAPI 入口
│       ├── agent.py          # Agent 核心逻辑
│       ├── models.py         # Pydantic 模型
│       ├── tools.py          # 工具定义
│       ├── config.py         # 配置管理
│       └── api/
│           ├── __init__.py
│           └── routes.py     # API 路由
└── tests/
    ├── __init__.py
    ├── conftest.py
    ├── test_tools.py
    ├── test_agent.py
    └── test_api.py
```

> **`__init__.py` 是什么？**
> Python 用它标记一个目录是"包"（package）。没有它，`from mini_agent import agent` 会报错。
> 文件可以为空，但必须存在（Python 3.3+ 有 namespace packages 可以省略，但不推荐）。
> JS 对比：Node.js 的 `package.json`，但 `__init__.py` 可以放初始化代码。

> **为什么用 `src/` 布局？**
> 防止 `import mini_agent` 意外地导入当前目录的代码（而非安装后的包）。
> `pip install -e .` 会把 `src/mini_agent` 安装到 site-packages，确保 import 的是正确的包。

> **`if __name__ == "__main__":` 是什么？**
> Python 文件被直接运行时，`__name__` 等于 `"__main__"`；被 import 时，`__name__` 等于模块名。
> 这个模式让文件既能被直接运行（执行测试代码），又能被 import（不执行测试代码）。
> ```python
> # calculator.py
> def add(a, b): return a + b
>
> if __name__ == "__main__":
>     print(add(1, 2))  # 直接运行 python calculator.py 时执行
>     # 被 import 时不会执行
> ```
> JS 对比：类似 Node.js 的 `if (require.main === module)`，但 JS 很少用这个模式。

## 二、pyproject.toml

> **JS 对比：**
> - `pyproject.toml` ≈ `package.json`（项目元数据 + 依赖声明）
> - `pip install -e ".[dev]"` ≈ `npm install`（安装依赖）
> - `pip freeze > requirements.txt` ≈ `package-lock.json`（锁定版本）
> - `setuptools` ≈ `npm` 的包管理器角色

```toml
[project]
name = "mini-agent"
version = "0.1.0"
requires-python = ">=3.11"
dependencies = [
    "openai>=1.60.0",
    "pydantic>=2.0",
    "pydantic-settings>=2.0",    # 环境变量管理（类似 dotenv + 校验）
    "fastapi>=0.115.0",
    "uvicorn>=0.30.0",           # ASGI 服务器（类似 Node.js 的 http.server）
    "httpx>=0.27.0",             # HTTP 客户端（类似 fetch）
    "python-dotenv>=1.0.0",      # 读取 .env 文件
]

[project.optional-dependencies]
dev = [
    "pytest>=8.0",
    "pytest-asyncio>=0.23",
    "pytest-cov>=5.0",
    "respx>=0.21",
]
```

## 三、config.py —— 配置管理

```python
from pydantic_settings import BaseSettings

class Settings(BaseSettings):
    openai_api_key: str
    openai_model: str = "gpt-4o"
    openai_base_url: str = "https://api.openai.com/v1"
    max_agent_steps: int = 10
    log_level: str = "INFO"

    model_config = {"env_file": ".env"}

settings = Settings()
```

## 四、models.py —— 数据模型

```python
from pydantic import BaseModel, Field
from enum import Enum

class Role(str, Enum):
    USER = "user"
    ASSISTANT = "assistant"
    SYSTEM = "system"
    TOOL = "tool"

class Message(BaseModel):
    role: Role
    content: str
    tool_call_id: str | None = None

class ChatRequest(BaseModel):
    message: str = Field(min_length=1, max_length=10000)
    thread_id: str | None = None

class ChatResponse(BaseModel):
    reply: str
    tool_calls: list[dict] = []
    steps: int = 0

class StreamEvent(BaseModel):
    type: str  # "thinking" | "tool_call" | "token" | "done" | "error"
    content: str = ""
    name: str = ""
```

## 五、tools.py —— 工具定义

```python
from langchain.tools import tool
import math

@tool
def calculate(expression: str) -> str:
    """计算数学表达式。支持基本运算和 math 函数。

    Args:
        expression: 数学表达式，如 "2+3*4" 或 "math.sqrt(16)"
    """
    try:
        allowed = {"math": math, "abs": abs, "round": round, "int": int, "float": float}
        result = eval(expression, {"__builtins__": {}}, allowed)
        return str(result)
    except Exception as e:
        return f"计算错误: {e}"

@tool
def search_knowledge(query: str) -> str:
    """搜索知识库，返回相关信息。

    Args:
        query: 搜索关键词
    """
    # Mock 知识库
    knowledge = {
        "Python": "Python 是一种高级编程语言，由 Guido van Rossum 于 1991 年创建。",
        "AI": "人工智能(AI)是计算机科学的一个分支，致力于创建能模拟人类智能的系统。",
        "Agent": "AI Agent 是能自主执行任务的智能体，通常结合 LLM、工具和记忆。",
    }
    for key, value in knowledge.items():
        if key.lower() in query.lower():
            return value
    return f"未找到关于 '{query}' 的信息"

@tool
def get_current_time() -> str:
    """获取当前日期和时间"""
    from datetime import datetime
    return datetime.now().strftime("%Y-%m-%d %H:%M:%S")

TOOLS = [calculate, search_knowledge, get_current_time]
```
