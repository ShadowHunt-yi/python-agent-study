## 关键概念速查

| 概念 | Python | JavaScript |
|------|--------|-----------|
| 类型注解 | `x: int` | TypeScript `x: number` |
| 不可变默认值 | `tuple()` | `Object.freeze([])` |
| 装饰器 | `@decorator` | 高阶函数/HOC |
| 异步 | `async/await` | `async/await` |
| 上下文管理 | `with` | `try/finally` |
| 生成器 | `yield` | `function*` |
| 列表推导 | `[x for x in ...]` | `.map()` |
| 字典推导 | `{k: v for ...}` | `Object.fromEntries()` |
