# 06 — 推导式（Comprehensions）

## 一、列表推导式

```python
# 基础
squares = [x**2 for x in range(10)]
# [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

# 带过滤
even_squares = [x**2 for x in range(10) if x % 2 == 0]
# [0, 4, 16, 36, 64]

# 带 if/else（注意顺序！）
labels = ["even" if x % 2 == 0 else "odd" for x in range(5)]
# ['even', 'odd', 'even', 'odd', 'even']

# 嵌套循环
matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
flat = [x for row in matrix for x in row]
# [1, 2, 3, 4, 5, 6, 7, 8, 9]

# 嵌套推导
transposed = [[row[i] for row in matrix] for i in range(3)]
# [[1, 4, 7], [2, 5, 8], [3, 6, 9]]
```

```javascript
// JS 对比
const squares = Array.from({ length: 10 }, (_, i) => i ** 2);
const evenSquares = Array.from({ length: 10 }, (_, i) => i)
    .filter(x => x % 2 === 0)
    .map(x => x ** 2);
const flat = matrix.flat();
```

### 顺序规则（面试常考）

```python
# 条件值：if/else 在 for 前面
[值_if_True if 条件 else 值_if_False for x in 序列]

# 过滤：if 在 for 后面
[值 for x in 序列 if 条件]
```

```python
# 两者结合
results = [
    x if x > 0 else 0     # 条件值
    for x in numbers
    if x != 0              # 过滤
]
```

### 什么时候不该用推导式

```python
# ✗ 推导式太复杂时，用普通循环更清晰
result = [
    process(x, y)
    for x in data
    for y in x.items()
    if y[1] > threshold
    and y[0] not in exclude_list
]

# ✓ 改用普通循环
result = []
for x in data:
    for y in x.items():
        if y[1] > threshold and y[0] not in exclude_list:
            result.append(process(x, y))
```

## 二、字典推导式

```python
# 基础
squares = {x: x**2 for x in range(5)}
# {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

# 从两个列表创建
keys = ["a", "b", "c"]
values = [1, 2, 3]
mapping = {k: v for k, v in zip(keys, values)}
# {'a': 1, 'b': 2, 'c': 3}

# 过滤
original = {"a": 1, "b": 2, "c": 3, "d": 0}
filtered = {k: v for k, v in original.items() if v > 0}
# {'a': 1, 'b': 2, 'c': 3}

# 反转字典
inverted = {v: k for k, v in original.items()}

# 转换值
upper = {k: v.upper() for k, v in {"name": "alice", "city": "nyc"}.items()}
# {'name': 'ALICE', 'city': 'NYC'}
```

## 四、生成器表达式

```python
# 类似列表推导式，但是惰性求值（不立即计算）
squares_list = [x**2 for x in range(1000000)]   # 立即占用内存
squares_gen  = (x**2 for x in range(1000000))   # 几乎不占内存

# 生成器只能遍历一次
gen = (x**2 for x in range(5))
list(gen)   # [0, 1, 4, 9, 16]
list(gen)   # [] —— 已经耗尽了

# 直接传给聚合函数（不需要额外括号）
total = sum(x**2 for x in range(100))     # 单参数可省略外层括号
maximum = max(x**2 for x in range(100))
joined = ",".join(str(x) for x in range(5))
```

### 生成器 vs 列表推导式

```python
import sys

list_comp = [x**2 for x in range(10000)]
gen_exp   = (x**2 for x in range(10000))

sys.getsizeof(list_comp)   # ~87000+ 字节
sys.getsizeof(gen_exp)     # ~120 字节 —— 极省内存
```

### 何时用生成器

```python
# 1. 数据量大，不需要全部在内存中
def process_large_file(path):
    lines = (line.strip() for line in open(path))  # 惰性读取
    return sum(1 for line in lines if line)         # 计算非空行数

# 2. 链式处理（管道模式）
data = range(1000000)
filtered = (x for x in data if x % 2 == 0)
mapped = (x**2 for x in filtered)
result = sum(mapped)    # 一次遍历完成所有操作

# 3. 只需要部分结果
from itertools import islice
first_10 = list(islice((x**2 for x in range(1000000)), 10))
# [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]
```

## 五、嵌套推导式实战

```python
# 矩阵转置
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9],
]

transposed = [[row[i] for row in matrix] for i in range(3)]
# [[1, 4, 7], [2, 5, 8], [3, 6, 9]]

# 展平嵌套列表
nested = [[1, 2], [3, 4, 5], [6]]
flat = [item for sublist in nested for item in sublist]
# [1, 2, 3, 4, 5, 6]

# 字典分组
students = [
    {"name": "Alice", "grade": "A"},
    {"name": "Bob", "grade": "B"},
    {"name": "Charlie", "grade": "A"},
]

from collections import defaultdict
groups = defaultdict(list)
[groups[s["grade"]].append(s["name"]) for s in students]
# {'A': ['Alice', 'Charlie'], 'B': ['Bob']}
```

## 六、面试高频问题

**Q: 列表推导式和 map/filter 哪个好？**
A: 推导式更 Pythonic、更可读。简单场景用推导式，复杂函数组合用 map/filter。

**Q: 生成器表达式和列表推导式的区别？**
A: 生成器惰性求值、省内存、只能遍历一次。列表推导式立即求值、占内存、可重复遍历。

**Q: 推导式中 if/else 放哪里？**
A: 条件值（if/else）在 for 前面，过滤（if）在 for 后面。两者不能混用同一个 if。
