# 02 — 控制流

## 一、if/elif/else

```python
temperature = 35

if temperature > 100:
    print("Too hot")
elif temperature > 75:       # 注意：是 elif，不是 else if
    print("Warm")
elif temperature > 50:
    print("Cool")
else:
    print("Cold")
```

```javascript
// JS 对比
if (temperature > 100) {
    console.log("Too hot");
} else if (temperature > 75) {  // else if 两个词
    console.log("Warm");
} else if (temperature > 50) {
    console.log("Cool");
} else {
    console.log("Cold");
}
```

### 关键差异

| 差异 | Python | JS |
|------|--------|-----|
| 关键字 | `elif` | `else if` |
| 条件括号 | 不需要 | 需要 `()` |
| 代码块 | 缩进（4 空格） | 花括号 `{}` |
| 三元表达式 | `x if cond else y` | `cond ? x : y` |

### 三元表达式

```python
# Python
status = "adult" if age >= 18 else "minor"

# JS
const status = age >= 18 ? "adult" : "minor";
```

### 条件表达式中的真值判断

```python
# Pythonic 写法 —— 利用 falsy 值
name = ""
display = name or "Anonymous"    # "Anonymous"

items = []
if not items:                    # 空列表为 falsy
    print("No items")

# 非 Pythonic（但能用）
if len(items) == 0:              # 不推荐
    print("No items")
if items == []:                  # 不推荐
    print("No items")
```

## 二、for 循环

Python 的 `for` **永远是 for..of 风格**，没有 C 风格的 `for (let i = 0; i < n; i++)`。

```python
# 基础遍历
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# 带索引 —— 用 enumerate()
for index, fruit in enumerate(fruits):
    print(f"{index}: {fruit}")  # f"..." 格式化字符串，{表达式} 会自动求值并插入，详见 05-字符串格式化.md
# → 0: apple
# → 1: banana
# → 2: cherry

# enumerate 可指定起始编号
for index, fruit in enumerate(fruits, start=1):
    print(f"{index}. {fruit}")
# → 1. apple
# → 2. banana
# → 3. cherry
```

```javascript
// JS 对比
for (const fruit of fruits) {
    console.log(fruit);
}

fruits.forEach((fruit, index) => {
    console.log(`${index}: ${fruit}`);
});
```

### range() —— 数字序列

```python
range(5)          # 0, 1, 2, 3, 4
range(2, 10)      # 2, 3, 4, 5, 6, 7, 8, 9
range(0, 10, 2)   # 0, 2, 4, 6, 8
range(10, 0, -1)  # 10, 9, 8, ..., 1

# 常见用法
for i in range(len(fruits)):   # 不推荐，直接用 enumerate
    print(fruits[i])

for i in range(5):
    print(i)
```

```javascript
// JS 对比
for (let i = 0; i < 5; i++) { ... }
Array.from({ length: 5 }, (_, i) => i);
```

### 同时遍历多个序列 —— zip()

```python
names = ["Alice", "Bob", "Charlie"]
ages = [25, 30, 35]
cities = ["NYC", "LA", "SF"]

for name, age, city in zip(names, ages, cities):
    print(f"{name}, {age}, from {city}")
# → Alice, 25, from NYC
# → Bob, 30, from LA
# → Charlie, 35, from SF

# zip 以最短的为准
# 如果要以最长的为准，用 itertools.zip_longest
```

```javascript
// JS 对比
names.forEach((name, i) => {
    console.log(`${name}, ${ages[i]}, from ${cities[i]}`);
});
```

### 遍历字典

```python
user = {"name": "Alice", "age": 30, "city": "NYC"}

# 遍历 key
for key in user:
    print(key)

# 遍历 value
for value in user.values():
    print(value)

# 遍历 key-value
for key, value in user.items():
    print(f"{key}: {value}")
# → name: Alice
# → age: 30
# → city: NYC
```

```javascript
// JS
for (const [key, value] of Object.entries(user)) {
    console.log(`${key}: ${value}`);
}
```

## 四、break / continue / pass

```python
# break —— 跳出循环
for n in range(10):
    if n == 7:
        break
    print(n)       # 0, 1, 2, 3, 4, 5, 6

# continue —— 跳过当前迭代
for n in range(10):
    if n % 2 == 0:
        continue
    print(n)       # 1, 3, 5, 7, 9

# pass —— 占位符，什么都不做
for n in range(10):
    pass           # TODO: 实现 later
```

## 五、循环的 else 子句（Python 独有）

```python
# else 在循环正常结束（没有 break）时执行
for n in range(2, 10):
    for x in range(2, n):
        if n % x == 0:
            break
    else:
        # 循环没有 break，说明 n 是质数
        print(f"{n} is prime")
# → 2 is prime
# → 3 is prime
# → 5 is prime
# → 7 is prime
```

```python
# 实际用法：搜索未找到时的处理
target = 42
items = [1, 5, 10, 20]

for item in items:
    if item == target:
        print("Found!")
        break
else:
    print("Not found")  # 循环正常结束，没有 break
```

## 六、match 语句（Python 3.10+）

比 JS 的 `switch` 强大得多，支持结构化模式匹配。

### 基础值匹配

```python
def http_status(status: int) -> str:
    match status:
        case 200:
            return "OK"
        case 400:
            return "Bad Request"
        case 401 | 403:          # OR 模式
            return "Unauthorized"
        case 404:
            return "Not Found"
        case _:                  # 通配符（默认）
            return f"Unknown: {status}"
```

```javascript
// JS switch —— 功能弱很多
function httpStatus(status) {
    switch (status) {
        case 200: return "OK";
        case 400: return "Bad Request";
        case 401:
        case 403: return "Unauthorized";
        case 404: return "Not Found";
        default:  return `Unknown: ${status}`;
    }
}
```

### 结构化模式匹配（Python 独有强大功能）

```python
# 匹配元组 —— 解构
def process_command(command: tuple) -> str:
    match command:
        case ("quit",):
            return "退出"
        case ("move", x, y):
            return f"移动到 ({x}, {y})"
        case ("attack", target) if target != "self":
            return f"攻击 {target}"
        case ("say", *words):
            return f"说: {' '.join(words)}"
        case _:
            return "未知命令"

process_command(("move", 10, 20))      # → "移动到 (10, 20)"
process_command(("say", "hello", "world"))  # → "说: hello world"
```

```python
# 匹配字典
def process_event(event: dict) -> str:
    match event:
        case {"type": "click", "x": x, "y": y}:
            return f"点击 ({x}, {y})"
        case {"type": "keypress", "key": key}:
            return f"按键: {key}"
        case {"type": "scroll", **rest}:
            return f"滚动: {rest}"
        case _:
            return "未知事件"

# 用法示例：
# process_event({"type": "click", "x": 10, "y": 20})  → "点击 (10, 20)"
# process_event({"type": "keypress", "key": "Enter"})  → "按键: Enter"
# process_event({"type": "scroll", "delta": 100})       → "滚动: {'delta': 100}"
```

```python
# 匹配类实例（class 语法详见 Day02-类与面向对象）
class Circle:
    def __init__(self, radius):
        self.radius = radius

class Rectangle:
    def __init__(self, width, height):
        self.width = width
        self.height = height

def area(shape) -> float:
    import math  # import 导入模块，详见 09-文件与模块.md
    match shape:
        case Circle(radius=r):
            return math.pi * r ** 2
        case Rectangle(width=w, height=h):
            return w * h
        case _:
            raise ValueError(f"未知形状: {shape}")

# 用法示例：
# area(Circle(5))             → 78.53981633974483
# area(Rectangle(4, 6))       → 24
```

### 守卫条件（Guard）

```python
match point:
    case Point(x, y) if x == y:
        print("在对角线上")
    case Point(x, y) if x == 0:
        print("在 Y 轴上")
    case Point(x, y):
        print(f"在 ({x}, {y})")

# 假设 point = Point(3, 3) → "在对角线上"
# 假设 point = Point(0, 5) → "在 Y 轴上"
# 假设 point = Point(2, 7) → "在 (2, 7)"
```

### 模式匹配陷阱

```python
# 变量在模式中是绑定（捕获），不是比较！
RED = "red"

match color:
    case RED:        # ✗ 错误 —— 这会把任何值绑定到变量 RED
        print("红色")
    case "red":      # ✓ 正确 —— 匹配字面量 "red"
        print("红色")
```

## 七、列表推导式中的控制流

```python
# 条件值（if/else 在 for 前面）
labels = ["偶数" if x % 2 == 0 else "奇数" for x in range(5)]
# ['偶数', '奇数', '偶数', '奇数', '偶数']

# 过滤（if 在 for 后面）
evens = [x for x in range(10) if x % 2 == 0]
# [0, 2, 4, 6, 8]
```

## 八、面试高频问题

**Q: Python 的 for 和 JS 的 for 有什么区别？**
A: Python 的 for 只有 for..in 形式，等价于 JS 的 for..of。没有 C 风格的 `for(i=0;i<n;i++)`，用 `range()` 替代。

**Q: `break` 后的 `else` 什么时候执行？**
A: 只有循环正常结束（没有触发 `break`）时才执行。如果 `break` 了，`else` 不执行。

**Q: match 和 switch 的区别？**
A: Python match 支持结构化模式匹配（解构元组、字典、类实例），JS switch 只能匹配值。

**Q: 为什么 Python 没有 `++`？**
A: Python 的设计哲学是显式优于隐式。`count += 1` 比 `count++` 更清晰。Python 中整数是不可变对象，`++` 会产生误导。
