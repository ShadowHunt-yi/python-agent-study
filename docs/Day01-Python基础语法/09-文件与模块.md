# 09 — 文件与模块

## 一、文件操作

```python
# 读取文件
with open("data.txt", "r", encoding="utf-8") as f:
    content = f.read()         # 全部内容
    lines = f.readlines()      # 行列表

# 逐行读取（内存友好）
with open("large.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())

# 写入文件
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("Hello\n")
    f.write("World\n")

# 追加
with open("log.txt", "a", encoding="utf-8") as f:
    f.write("新日志\n")
```

```javascript
// Node.js 对比
const fs = require("fs").promises;

const content = await fs.readFile("data.txt", "utf-8");
const lines = content.split("\n");

await fs.writeFile("output.txt", "Hello\n");
await fs.appendFile("log.txt", "新日志\n");
```

## 二、pathlib（推荐）

```python
from pathlib import Path

# 创建路径
p = Path("data") / "users" / "alice.txt"   # data/users/alice.txt

# 常用操作
p.exists()           # 是否存在
p.is_file()          # 是否是文件
p.is_dir()           # 是否是目录
p.name               # "alice.txt"
p.stem               # "alice"
p.suffix             # ".txt"
p.parent             # Path("data/users")

# 读写
p.read_text(encoding="utf-8")
p.write_text("内容", encoding="utf-8")

# 遍历目录
for f in Path("data").glob("*.txt"):
    print(f)

# 递归遍历
for f in Path("data").rglob("*.py"):
    print(f)

# 创建目录
Path("output").mkdir(parents=True, exist_ok=True)
```

## 三、JSON 操作

```python
import json

# 读取
with open("config.json", "r", encoding="utf-8") as f:
    config = json.load(f)

# 写入
with open("output.json", "w", encoding="utf-8") as f:
    json.dump({"name": "Alice"}, f, ensure_ascii=False, indent=2)

# 字符串转换
text = json.dumps({"a": 1}, ensure_ascii=False)
data = json.loads('{"a": 1}')
```

```javascript
// JS 对比
const data = JSON.parse(fs.readFileSync("config.json", "utf-8"));
fs.writeFileSync("output.json", JSON.stringify(data, null, 2));
```

## 六、sys.argv —— 命令行参数

```python
import sys

# python script.py arg1 arg2
print(sys.argv)    # ['script.py', 'arg1', 'arg2']
```

## 七、面试高频问题

**Q: pathlib 和 os.path 的区别？**
A: pathlib 是面向对象的，更 Pythonic。os.path 是函数式的。推荐 pathlib。

**Q: `with open()` 的好处？**
A: 自动关闭文件，即使发生异常。避免资源泄漏。

**Q: 怎么处理大文件？**
A: 逐行迭代（`for line in f`），不要 `f.read()` 一次读入内存。
