# 05 — 字符串格式化

## 一、f-string（Python 3.6+）—— 推荐

```python
name = "Alice"
age = 30
score = 95.678

# 基础
print(f"Hello, {name}! You are {age} years old.")

# 表达式
print(f"Next year you'll be {age + 1}.")
print(f"Name uppercase: {name.upper()}")

# 格式化数字
print(f"Score: {score:.2f}")          # 95.68（2位小数）
print(f"Large: {1000000:,}")          # 1,000,000（千位分隔）
print(f"Percent: {0.856:.1%}")        # 85.6%
print(f"Hex: {255:#x}")              # 0xff
print(f"Binary: {10:#010b}")         # 0b00001010

# 对齐
print(f"{'left':<20}")               # 左对齐
print(f"{'right':>20}")              # 右对齐
print(f"{'center':^20}")             # 居中
print(f"{'pad':*^20}")               # 用 * 填充

# 嵌套引号
print(f"{'hello':>10}")              # 可以在 f-string 内用同种引号
```

```javascript
// JS 模板字符串
console.log(`Hello, ${name}! You are ${age} years old.`);
console.log(`Next year you'll be ${age + 1}.`);
// JS 没有内置格式化，需要手动处理
console.log(`Score: ${score.toFixed(2)}`);
```

### f-string 调试模式（Python 3.8+）

```python
x = 42
y = "hello"
print(f"{x=}")              # x=42
print(f"{x=}, {y=}")       # x=42, y='hello'
print(f"{x + 1=}")         # x + 1=43
```

## 二、.format() 方法

```python
# 位置参数
"Hello, {}!".format(name)
"Hello, {0}! Age: {1}".format(name, age)
"{0} {1} {0}".format("hello", "world")   # "hello world hello"

# 关键字参数
"Hello, {name}! Age: {age}".format(name="Alice", age=30)

# 格式化
"Score: {:.2f}".format(95.678)
"{:>10}".format("right")

# 访问属性和索引
user = {"name": "Alice", "scores": [90, 85, 95]}
"{0[name]} scored {0[scores][0]}".format(user)   # "Alice scored 90"
```

## 四、Template 字符串

```python
from string import Template

t = Template("Hello, $name! Age: $age")
t.substitute(name="Alice", age=30)         # 有缺失 key 会 KeyError
t.safe_substitute(name="Alice")            # 缺失 key 保留 $age
```

## 五、字符串常用方法

```python
s = "  Hello, World!  "

# 大小写
s.upper()               # "  HELLO, WORLD!  "
s.lower()               # "  hello, world!  "
s.title()               # "  Hello, World!  "
s.capitalize()          # "  hello, world!  "

# 去空白
s.strip()               # "Hello, World!"（去两端）
s.lstrip()              # "Hello, World!  "（去左端）
s.rstrip()               # "  Hello, World!"（去右端）

# 查找
s.strip().find("World") # 7（索引，找不到返回 -1）
s.strip().index("World") # 7（索引，找不到抛 ValueError）
s.strip().count("l")    # 3
s.strip().startswith("Hello")  # True
s.strip().endswith("!")        # True

# 替换
s.strip().replace("World", "Python")   # "Hello, Python!"

# 分割/合并
"a,b,c".split(",")           # ['a', 'b', 'c']
"a,b,c".split(",", 1)       # ['a', 'b,c']（限制分割次数）
"  a  b  c  ".split()        # ['a', 'b', 'c']（默认按空白分割）
",".join(["a", "b", "c"])    # "a,b,c"

# 判断
"hello".isalpha()            # True（全是字母）
"123".isdigit()              # True（全是数字）
"hello123".isalnum()         # True（字母或数字）
"".isspace()                 # False
"".strip() or "default"      # "default"（空字符串是 falsy）
```


## 八、面试高频问题

**Q: f-string 和 .format() 哪个好？**
A: f-string 更快（编译时处理）、更可读。Python 3.6+ 优先用 f-string。

**Q: Python 字符串是可变的吗？**
A: 不可变。所有"修改"操作都返回新字符串。

**Q: 如何高效拼接大量字符串？**
A: 用 `"".join(list)` 而不是 `+=` 循环拼接。`join` 是 O(n)，`+=` 循环是 O(n^2)。
