# 01 — FastAPI 基础路由与请求

## 一、最小应用

```python
from fastapi import FastAPI

app = FastAPI(title="My API", version="1.0.0")

@app.get("/")
async def root():
    return {"message": "Hello World"}

# 启动：uvicorn main:app --reload --port 8000
```

## 二、路径参数与查询参数

```python
from fastapi import FastAPI, Path, Query

app = FastAPI()

# 路径参数
@app.get("/users/{user_id}")
async def get_user(user_id: int = Path(description="用户ID")):
    return {"user_id": user_id}

# 查询参数
@app.get("/users/")
async def list_users(
    page: int = Query(1, ge=1, description="页码"),
    size: int = Query(20, ge=1, le=100, description="每页数量"),
    keyword: str | None = Query(None, description="搜索关键词"),
):
    return {"page": page, "size": size, "keyword": keyword}
```

### 参数优先级规则

```
路径中的参数 → 路径参数
函数签名中的 Pydantic Model → 请求体
其他 → 查询参数
```

## 三、请求体（Pydantic）

```python
from fastapi import FastAPI
from pydantic import BaseModel, Field

class ChatRequest(BaseModel):
    message: str = Field(min_length=1, max_length=10000)
    model: str = Field(default="gpt-4o")
    temperature: float = Field(default=0.7, ge=0, le=2)
    stream: bool = False

class ChatResponse(BaseModel):
    reply: str
    tokens_used: int

@app.post("/chat", response_model=ChatResponse)
async def chat(req: ChatRequest):
    return ChatResponse(reply="Hello!", tokens_used=10)
```

## 四、响应模型

```python
from fastapi import FastAPI
from pydantic import BaseModel

class UserOut(BaseModel):
    id: int
    name: str
    email: str
    # 不返回 password 等敏感字段

class UserIn(BaseModel):
    name: str
    email: str
    password: str

@app.post("/users", response_model=UserOut)
async def create_user(user: UserIn):
    # 保存到数据库...
    return {"id": 1, "name": user.name, "email": user.email}

# response_model 自动：
# 1. 过滤掉不在 UserOut 中的字段
# 2. 验证输出数据
# 3. 生成 OpenAPI 文档
```

## 五、状态码与异常

```python
from fastapi import FastAPI, HTTPException, status

app = FastAPI()

@app.get("/users/{user_id}", status_code=status.HTTP_200_OK)
async def get_user(user_id: int):
    if user_id <= 0:
        raise HTTPException(
            status_code=status.HTTP_404_NOT_FOUND,
            detail="用户不存在",
        )
    return {"id": user_id}
```

## 六、APIRouter —— 模块化路由

```python
# routers/users.py
from fastapi import APIRouter

router = APIRouter(prefix="/users", tags=["用户"])

@router.get("/{user_id}")
async def get_user(user_id: int):
    return {"user_id": user_id}

@router.post("/")
async def create_user(name: str):
    return {"name": name}

# main.py
from fastapi import FastAPI
from routers import users

app = FastAPI()
app.include_router(users.router)
```

## 八、面试高频问题

**Q: FastAPI 的优势是什么？**
A: 基于 Pydantic 自动校验、基于类型提示自动生成文档、原生异步支持、依赖注入系统。

**Q: 路径参数和查询参数怎么区分？**
A: 路径中的 `{param}` 是路径参数。函数签名中不在路径里的参数是查询参数。Pydantic Model 是请求体。

**Q: response_model 的作用？**
A: 过滤敏感字段、验证输出数据、自动生成 OpenAPI 文档。
