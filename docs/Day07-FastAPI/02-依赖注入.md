# 02 — 依赖注入

## 一、基础依赖

```python
from fastapi import FastAPI, Depends, Header, HTTPException

app = FastAPI()

async def get_current_user(authorization: str = Header(...)):
    """从 Header 中提取并验证用户"""
    if not authorization.startswith("Bearer "):
        raise HTTPException(status_code=401, detail="无效的认证信息")
    token = authorization[7:]
    # 验证 token...
    return {"user_id": "123", "name": "Alice"}

@app.get("/me")
async def get_me(user: dict = Depends(get_current_user)):
    return user
```

## 二、类依赖

```python
from fastapi import Depends

class Pagination:
    def __init__(self, page: int = 1, size: int = 20):
        self.page = max(1, page)
        self.size = min(100, max(1, size))
        self.offset = (self.page - 1) * self.size

@app.get("/items")
async def list_items(pagination: Pagination = Depends()):
    return {"page": pagination.page, "size": pagination.size}
```

## 三、依赖链

```python
async def get_db():
    db = SessionLocal()
    try:
        yield db
    finally:
        db.close()

async def get_current_user(
    authorization: str = Header(...),
    db: Session = Depends(get_db),
):
    user = db.query(User).filter(User.token == authorization).first()
    if not user:
        raise HTTPException(status_code=401)
    return user

@app.get("/orders")
async def list_orders(
    user: dict = Depends(get_current_user),  # 依赖链
    db: Session = Depends(get_db),
):
    return db.query(Order).filter(Order.user_id == user["user_id"]).all()
```

## 四、yield 依赖（资源管理）

```python
from fastapi import Depends

async def get_db():
    db = SessionLocal()
    try:
        yield db       # 这里是注入点
    finally:
        db.close()     # 请求结束后清理

# 或者用 contextmanager
from contextlib import asynccontextmanager

@asynccontextmanager
async def get_db():
    db = SessionLocal()
    try:
        yield db
    finally:
        db.close()
```

## 五、全局依赖

```python
app = FastAPI(dependencies=[Depends(get_current_user)])
# 所有路由都会自动验证用户
```

## 六、Overriding（测试用）

```python
# 测试时替换依赖
async def override_get_db():
    yield test_db

app.dependency_overrides[get_db] = override_get_db
```

## 七、面试高频问题

**Q: Depends 的作用？**
A: 声明依赖关系，FastAPI 自动解析并注入。支持依赖链、yield 资源管理、全局依赖。

**Q: yield 依赖和普通依赖的区别？**
A: yield 依赖在注入后执行清理代码（如关闭数据库连接）。普通依赖只返回值。

**Q: 怎么测试需要认证的接口？**
A: 用 `app.dependency_overrides` 替换依赖函数。
