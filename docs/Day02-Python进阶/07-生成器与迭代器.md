# 07 — 生成器与迭代器

## 一、迭代器协议

```python
# 可迭代对象 —— 有 __iter__ 方法
# 迭代器 —— 有 __iter__ 和 __next__ 方法

# 手动迭代
nums = [1, 2, 3]
it = iter(nums)       # 获取迭代器
next(it)              # 1
next(it)              # 2
next(it)              # 3
next(it)              # StopIteration

# for 循环内部就是调用 iter() 和 next()
for x in nums:
    print(x)
```

## 二、生成器函数

用 `yield` 代替 `return`，每次调用 `next()` 执行到下一个 `yield`。

```python
def countdown(n):
    """从 n 倒数到 1"""
    while n > 0:
        yield n
        n -= 1

for num in countdown(5):
    print(num)    # 5, 4, 3, 2, 1

# 手动迭代
gen = countdown(3)
next(gen)   # 3
next(gen)   # 2
next(gen)   # 1
next(gen)   # StopIteration
```

```javascript
// JS 对比 —— function* + yield
function* countdown(n) {
    while (n > 0) {
        yield n--;
    }
}
```

## 四、yield from —— 委托生成器

```python
def flatten(nested):
    """展平嵌套列表"""
    for item in nested:
        if isinstance(item, list):
            yield from flatten(item)   # 委托给子生成器
        else:
            yield item

list(flatten([1, [2, [3, 4], 5], 6]))
# [1, 2, 3, 4, 5, 6]
```

```python
# 链接多个可迭代对象
def chain(*iterables):
    for it in iterables:
        yield from it

list(chain([1, 2], [3, 4], [5, 6]))
# [1, 2, 3, 4, 5, 6]
```

## 五、.send() —— 向生成器发送数据

```python
def accumulator():
    total = 0
    while True:
        value = yield total
        if value is None:
            break
        total += value

gen = accumulator()
next(gen)          # 初始化（必须先调一次）
gen.send(10)       # 10
gen.send(20)       # 30
gen.send(5)        # 35
```

```python
# 实际用法：动态调整生成器行为
def running_average():
    total = 0
    count = 0
    average = None
    while True:
        value = yield average
        if value is None:
            break
        total += value
        count += 1
        average = total / count

avg = running_average()
next(avg)
avg.send(10)    # 10.0
avg.send(20)    # 15.0
avg.send(30)    # 20.0
```

## 六、管道模式

```python
# 生成器可以像 Unix 管道一样组合
def read_lines(path):
    """读取文件行"""
    with open(path, "r", encoding="utf-8") as f:
        for line in f:
            yield line.strip()

def filter_comments(lines):
    """过滤注释行"""
    for line in lines:
        if not line.startswith("#"):
            yield line

def filter_empty(lines):
    """过滤空行"""
    for line in lines:
        if line:
            yield line

def parse_csv(lines):
    """解析 CSV"""
    for line in lines:
        yield line.split(",")

# 组合成管道 —— 全程惰性，内存高效
lines = read_lines("data.csv")
non_comments = filter_comments(lines)
non_empty = filter_empty(non_empty)
records = parse_csv(non_empty)

for record in records:
    process(record)
```

## 七、itertools —— 标准库工具

```python
import itertools

# count —— 无限计数
for i in itertools.count(10):    # 10, 11, 12, ...
    if i > 15: break

# cycle —— 无限循环
for item in itertools.cycle(["a", "b", "c"]):
    print(item)    # a, b, c, a, b, c, ...

# chain —— 链接多个迭代器
list(itertools.chain([1, 2], [3, 4]))    # [1, 2, 3, 4]

# islice —— 切片生成器
list(itertools.islice(itertools.count(), 5))    # [0, 1, 2, 3, 4]

# groupby —— 分组
data = [("a", 1), ("a", 2), ("b", 3), ("b", 4)]
for key, group in itertools.groupby(data, key=lambda x: x[0]):
    print(key, list(group))

# product —— 笛卡尔积
list(itertools.product([1, 2], ["a", "b"]))
# [(1, 'a'), (1, 'b'), (2, 'a'), (2, 'b')]

# permutations —— 排列
list(itertools.permutations([1, 2, 3], 2))
# [(1, 2), (1, 3), (2, 1), (2, 3), (3, 1), (3, 2)]

# combinations —— 组合
list(itertools.combinations([1, 2, 3], 2))
# [(1, 2), (1, 3), (2, 3)]
```

def batch_process(items, batch_size=10):
    """将数据分批"""
    batch = []
    for item in items:
        batch.append(item)
        if len(batch) >= batch_size:
            yield batch
            batch = []
    if batch:

## 九、面试高频问题

**Q: 生成器和列表推导式的区别？**
A: 生成器惰性求值、省内存、只能遍历一次。列表推导式立即求值、可重复遍历。

**Q: `yield` 和 `return` 的区别？**
A: `return` 终止函数并返回值。`yield` 暂停函数并返回值，下次调用 `next()` 从暂停处继续。

**Q: 如何在 Agent 开发中使用生成器？**
A: 流式 LLM 输出、大数据集处理、批量 API 调用、管道式数据处理。

**Q: `yield from` 的作用？**
A: 将迭代委托给子生成器。简化嵌套 yield，如 `yield from flatten(item)` 等价于 `for x in flatten(item): yield x`。
