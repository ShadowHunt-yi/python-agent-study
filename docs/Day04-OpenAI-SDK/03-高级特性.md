## 五、对话历史管理

```python
class Conversation:
    def __init__(self, system_prompt: str = ""):
        self.messages: list[dict] = []
        if system_prompt:
            self.messages.append({"role": "system", "content": system_prompt})

    def add_user(self, content: str):
        self.messages.append({"role": "user", "content": content})

    def add_assistant(self, content: str, tool_calls=None):
        msg = {"role": "assistant", "content": content}
        if tool_calls:
            msg["tool_calls"] = tool_calls
        self.messages.append(msg)

    def add_tool_result(self, tool_call_id: str, content: str):
        self.messages.append({
            "role": "tool",
            "tool_call_id": tool_call_id,
            "content": content,
        })

    def trim(self, max_tokens: int = 4000):
        """裁剪历史，保留 system + 最近的消息"""
        # 保留 system message
        system = [m for m in self.messages if m["role"] == "system"]
        others = [m for m in self.messages if m["role"] != "system"]

        # 从最新的开始保留
        while others and self._count_tokens(system + others) > max_tokens:
            others.pop(0)

        self.messages = system + others

    def _count_tokens(self, messages: list[dict]) -> int:
        encoding = tiktoken.encoding_for_model("gpt-4o")
        total = 0
        for msg in messages:
            total += len(encoding.encode(msg.get("content", "")))
            total += 4  # 消息开销
        return total
```

## 八、异步重试策略

```python
import asyncio
from openai import AsyncOpenAI, RateLimitError, APIConnectionError

client = AsyncOpenAI(max_retries=0)  # 禁用内置重试，手动控制

async def call_with_retry(func, *args, max_retries=5, **kwargs):
    for attempt in range(max_retries):
        try:
            return await func(*args, **kwargs)
        except RateLimitError:
            wait = 2 ** attempt
            print(f"限流，{wait}s 后重试 ({attempt+1}/{max_retries})")
            await asyncio.sleep(wait)
        except APIConnectionError:
            wait = 2 ** attempt
            print(f"网络错误，{wait}s 后重试")
            await asyncio.sleep(wait)
    raise Exception("超过最大重试次数")
```

## 九、面试高频问题

**Q: 怎么处理 LLM 的 JSON 输出？**
A: 两种方式：1) `response_format={"type": "json_object"}` + 自行解析；2) `client.beta.chat.completions.parse()` + Pydantic 自动校验。

**Q: 并行工具调用怎么处理？**
A: LLM 一次返回多个 tool_calls，遍历执行后将所有结果作为 tool message 一起发回。

**Q: 怎么控制 Token 成本？**
A: 1) 用 tiktoken 预估；2) 裁剪对话历史；3) 用更小的模型；4) 设置 max_tokens。
